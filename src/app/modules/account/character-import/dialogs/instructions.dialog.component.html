How to export your character from another server

Instructions

1.) Download and install our Character Profiler addon for the version of the game you wish to export a character from. The addon can be downloaded from  here 2.) Log in to the server and ensure that the addon is enabled, you may have to check 'Load out of date Addons' 3.) Enter World with the character you want to import 4.) Type   /cp purge all   to make sure you have no dangling profiles 5.) Type   /cp export   to begin capturing a snapshot of your character 6.) Open your bank and open all bags 7.) Type   /cp export   again to capture a bank snapshot 8.) Log out 9.) Your character profile will be available in [Wow Directory]/WTF/Account/[accountname]/SavedVariables/CharacterProfiler.lua 10.) Upload the CharacterProfiler.lua file and all exported Rare, Epic and Legendary items will be mailed to your character of choice
